What is oxide?

oxide is a debian-linux based operating system for home camputers, oxide focuses on security, compatibility and ease of use

oxide uses kde as an desktop enviornment included with wine package for windows compatibility layer which allows you to run windows programs on oxide

oxide iso files will be released on oxide github page

Who is the creator of oxide?

Arshavir Mirzakhani is the main creator of oxide and its logo
